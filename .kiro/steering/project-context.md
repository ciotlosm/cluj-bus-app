# Project Context

## Product
Real-time bus tracking app using Tranzy API for live vehicle locations and schedules.

## Core Features
- Live vehicle tracking with confidence indicators
- Location-aware station display (home/work directions)
- Mobile-first Material Design interface
- Offline support via service worker

## Documentation Structure
- **All docs go in `docs/` folder** (never create .md files in root)
- **Use existing consolidated docs** (don't create new files)
- **Update changelog.md** for significant changes

## Development Focus
- **Reliability**: Proper error detection for network/GPS
- **Performance**: Intelligent caching and auto-refresh  
- **UX**: Clear confidence indicators and error handling
